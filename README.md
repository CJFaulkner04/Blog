# Spring Boot Blog Application

## Development Instructions

- `git clone https://github.com/CJFaulkner04/Blog.git`
- `cd Blog`
- open in favorite editor, or
- `mvnw spring-boot:run`
- open http://localhost:8080 and Blog away!
